# Python version: 3.9.x  (see note below)
#
# WHY Python 3.9?
#   • The source files use  X | None  and  tuple[X, Y]  union-type syntax
#     that was introduced in Python 3.10.  The fixed versions of layers.py,
#     model.py, vi.py and utils.py replace those hints with
#     Optional[X] / Tuple[X, Y] from `typing`, making the code fully
#     compatible with Python 3.9.
#   • PyTorch 2.x officially supports Python 3.9 – 3.12.
#   • Python 3.9 is the oldest version still receiving security patches
#     from the PSF, making it a safe long-term choice for embedded/research
#     workflows.
#
# Recommended setup (conda):
#   conda create -n snn python=3.9
#   conda activate snn
#   pip install -r requirements.txt
#
# Recommended setup (venv):
#   python3.9 -m venv .venv
#   source .venv/bin/activate          # Windows: .venv\Scripts\activate
#   pip install -r requirements.txt
# =============================================================================
